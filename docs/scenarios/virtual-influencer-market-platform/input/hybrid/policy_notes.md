# Hybrid Policy Notes

## Platforms in scope
- X
- Reddit
- Mainstream discovery platform
- Premium community endpoint

## Checklist

1. Platform conduct and content controls
- [x] Harassment and abuse policy reviewed — X: Community Notes + support block and mute; X now permits adult content in sensitive-media settings for 18+ accounts (2025 policy); moderation relies on user reporting; creator account protected-mode available if brigaded. Reddit: Subreddit-level moderation is primary; site-wide admin action for severe violations; creator posts can be removed by subreddit mods without creator appeal rights.
- [x] Synthetic media policy reviewed — X: No blanket ban on AI-generated personas; labelling encouraged but not enforced for entertainment accounts; Political AI content labels introduced 2024 for elections context only. Reddit: No site-wide AI persona ban; individual subreddit rules vary significantly — verify rules per target subreddit before posting; some communities explicitly ban AI-generated content.
- [x] Adult content boundary policy reviewed — X: Explicit content permitted with Sensitive Media flag enabled and 18+ account designation; non-explicit suggestive content allowed without restriction. Reddit: NSFW flair required per post; age verification gate shown to unlogged users; community-level rules may be stricter than site-wide policy.

2. Community operations
- [x] Moderation staffing assumptions documented — 1 community manager handles daily moderation across X and Reddit; 4-hour response SLA for standard incidents; strategist escalation required for brand or compliance-level calls; tool assumption: Moderation inbox aggregator (e.g., Hootsuite or Buffer with X/Reddit connectors).
- [x] Escalation process for raids or brigading documented — X: Temporary DM restriction + mass-report tool activation; private account mode as fallback if sustained attack persists beyond 48h. Reddit: Contact subreddit moderators directly via ModMail + escalate to r/reddit admins; maintain emergency private subreddit mirror for community continuity during attack.
- [x] Reputation incident response SLA documented — Tier 1 minor: 4-hour acknowledgement, 24-hour resolution target. Tier 2 press-visible: 1-hour response, senior team loop, pre-approved statement template deployed. Tier 3 viral crisis: immediate (<30 min) escalation, prepared dark-site holding statement, proactive platform outreach initiated.

3. Monetization controls
- [x] Chargeback handling assumptions documented — X Subscriptions and Stripe-billed premium community tiers: standard chargeback dispute SLA is 30 days; operator must respond with evidence of delivery; chargeback rate threshold >1% triggers Stripe account review and potential restriction; maintain 60-day receipts and access-log records.
- [x] Subscriber retention intervention plan documented — Day 45 inactivity: automated platform DM with exclusive content teaser. Day 55: personalised re-engagement post in community feed. Day 60: winback offer — one-month discount or bonus lore drop for at-risk subscribers. Churn beyond 90 days treated as lost; focus shifts to new acquisition.
- [x] Platform dependency fallback documented — Primary: X + Reddit discovery → premium community endpoint (Discord/Patreon). Fallback if X restricted or banned: Instagram Reels + Threads for reach replacement. Fallback if premium platform disrupted: email list (minimum 1,000 opt-in target as operational threshold) + Substack or Beehiiv mirror as newsletter fallback.
