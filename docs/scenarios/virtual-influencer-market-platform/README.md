# Virtual Influencer Market and Platform Scenario Pack

This documentation pack helps you execute a market-platform fit scenario in MiroFish-Offline for virtual influencers.

It is designed to compare platform strategies across:
- Demographic fit
- Visibility potential
- Growth velocity
- Monetization potential
- Compliance and reputation risk

It includes an adult-platform branch (for platforms such as OnlyFans) with strict 18+ and compliance controls.

## Files in this pack

- [Execution Playbook](./execution-playbook.md)
- [Data Requirements](./data-requirements.md)
- [Scoring Sheet CSV Template](./templates/platform-market-scoring-template.csv)
- [Risk Register CSV Template](./templates/risk-register-template.csv)
- [API Payload Templates](./templates/api-payloads.md)
- [Simulation Requirement Prompt Template](./templates/simulation-requirement-template.md)

## Fast start

1. Fill out [Data Requirements](./data-requirements.md).
2. Draft your scenario brief using [Simulation Requirement Prompt Template](./templates/simulation-requirement-template.md).
3. Run the scenario using [Execution Playbook](./execution-playbook.md).
4. Score outcomes in [Scoring Sheet CSV Template](./templates/platform-market-scoring-template.csv).
5. Track policy and brand risks in [Risk Register CSV Template](./templates/risk-register-template.csv).
6. Save key evidence with [Evidence Log Template](./templates/evidence-log-template.md).

## Scope and guardrails

- Adult-platform scenario branch is restricted to adult audiences only (18+).
- No strategy should rely on weak age-gating or ambiguous compliance.
- Any recommendation must include fallback channels in case monetization endpoints are disrupted.
