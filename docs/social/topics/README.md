# Topics Directory

## Overview

Content taxonomy for cross-platform analysis. This directory contains vertical and horizontal content classification to support MiroFish simulation.

## Files

| File | Description |
|------|-------------|
| `content-taxonomy.md` | Full taxonomy with verticals, themes, formats |

## Taxonomy Structure

- **Level 1**: Vertical categories (12 major verticals)
- **Level 2**: Content themes within verticals
- **Level 3**: Content formats and their characteristics

## Integration

- Maps to platform packets via vertical affinity
- Maps to personas via content preference
- Maps to journeys via content strategy
