## Cross-Platform Content Adaptation Guide

### TikTok → Instagram Reels

| TikTok Element | Instagram Adaptation |
|---------------|---------------------|
| Text overlay | Keep minimal, or carousel |
| Sound (trending) | Use IG trending audio |
| Hook (first 3 sec) | Same or extend to 5 sec |
| Fast pace | Slightly slower |
| Imperfect/raw | Can be more polished |
| Hashtags | Use @mentions instead |

### TikTok → YouTube Shorts

| TikTok Element | YouTube Adaptation |
|---------------|-------------------|
| Vertical 9:16 | Same aspect ratio |
| 15-60 sec | Can extend to 60 sec |
| Trending audio | Use YouTube Sound Library |
| Text overlay | Add captions (accessibility) |
| SEO hashtags | Title optimization |

### Instagram → YouTube Long-form

| Instagram Element | YouTube Adaptation |
|------------------|-------------------|
| Reels (60 sec) | Expand to 10-20 min |
| Quick tips | Full tutorials |
| Behind scenes | Vlog format |
| Grid aesthetic | Less critical |
